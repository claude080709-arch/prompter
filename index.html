기능 소개

왜 Prompter인가요

목표를 입력하는 것만으로, 전문가 수준의 결과를 얻을 수 있어요.

전문가 자동 매칭

목표를 입력하면 AI가 해당 분야에서 가장 효과적인 전문가 역할을 3~5개 추천해드려요.

맞춤형 프롬프트 생성

선택한 전문가를 기반으로 바로 ChatGPT, Claude에 붙여넣기 할 수 있는 완성된 프롬프트를 만들어줘요.

피드백 반영 재추천

추천된 전문가가 마음에 안 들면 피드백을 남기면 돼요. AI가 반영해서 새로운 전문가를 다시 추천해줘요.

복합 전문가 결합

여러 전문가를 동시에 선택하면 AI가 하나의 통합된 프롬프트로 자동으로 합쳐드려요.

즉시 사용 가능

설치 필요 없어요. 웹에서 바로 열고, 목표 입력 후 30초 안에 프롬프트를 받을 수 있어요.

영어 + 한국어 동시 제공

생성된 프롬프트는 영어와 한국어 번역본을 동시에 제공해요. 어떤 AI 서비스에서도 바로 쓸 수 있어요.

사용 방법

3단계면 충분해요

복잡한 설정 없이, 누구나 바로 쓸 수 있어요.

1
목표를 입력하세요

이루고 싶은 것을 자유롭게 적어주세요. "3개월 안에 10kg 빼고 싶어", "영어 회화를 빠르게 늘리고 싶어" 처럼 짧아도 괜찮아요.

2
전문가를 선택하세요

AI가 추천한 전문가 중 원하는 역할을 고르세요. 여러 명을 동시에 선택해도 돼요. 마음에 안 들면 피드백을 남겨 다시 추천받을 수 있어요.

3
프롬프트를 복사해서 쓰세요

생성된 영어 프롬프트를 복사해서 ChatGPT나 Claude에 붙여넣으면 끝이에요. 전문가 수준의 AI 답변을 바로 받을 수 있어요.
